.. Title

zscaler.ziacloud.zia_dc_exclusions module -- Manages ZIA DC (datacenter) exclusions
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `zscaler.ziacloud collection <https://galaxy.ansible.com/ui/repo/published/zscaler/ziacloud/>`_ (version 2.2.0).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible\-galaxy collection install zscaler.ziacloud`.
    You need further requirements to be able to use this module,
    see :ref:`Requirements <ansible_collections.zscaler.ziacloud.zia_dc_exclusions_module_requirements>` for details.

    To use it in a playbook, specify: :code:`zscaler.ziacloud.zia_dc_exclusions`.

.. version_added

.. rst-class:: ansible-version-added

New in zscaler.ziacloud 1.0.0

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- Creates, updates, or deletes datacenter exclusions for traffic forwarding.
- DC exclusions disable tunnels to specific Zscaler datacenters for a time period.

Requirements
------------
The below requirements are needed on the host that executes this module.

- Zscaler SDK Python can be obtained from PyPI \ `https://pypi.org/project/zscaler\-sdk\-python/ <https://pypi.org/project/zscaler-sdk-python/>`__

Notes
-----

.. note::
   - Check mode is supported.
   - Use :literal:`datacenter\_id` or :literal:`name` to reference an existing exclusion for update/delete.

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    - name: Create a DC exclusion
      zscaler.ziacloud.zia_dc_exclusions:
        provider: '{{ provider }}'
        datacenter_id: 5313
        start_time_utc: "04/29/2025 02:51:00 pm"
        end_time_utc: "05/01/2025 02:00:00 pm"
        description: "Maintenance window"

    - name: Update a DC exclusion by datacenter ID
      zscaler.ziacloud.zia_dc_exclusions:
        provider: '{{ provider }}'
        datacenter_id: 5313
        description: "Updated description"
        start_time: 1745941860
        end_time: 1746114660

    - name: Delete a DC exclusion
      zscaler.ziacloud.zia_dc_exclusions:
        provider: '{{ provider }}'
        datacenter_id: 5313
        state: absent
