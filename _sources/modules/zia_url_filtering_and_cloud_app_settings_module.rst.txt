.. note::
    This module is part of the `zscaler.ziacloud collection <https://galaxy.ansible.com/ui/repo/published/zscaler/ziacloud/>`_ (version 2.0.3).

    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    see :ref:`Requirements <ansible_collections.zscaler.ziacloud.zia_url_filtering_and_cloud_app_settings_module_requirements>` for details.
