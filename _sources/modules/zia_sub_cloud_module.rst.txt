.. Title

zscaler.ziacloud.zia_sub_cloud module -- Manages ZIA subcloud configurations
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `zscaler.ziacloud collection <https://galaxy.ansible.com/ui/repo/published/zscaler/ziacloud/>`_ (version 2.2.0).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible\-galaxy collection install zscaler.ziacloud`.
    You need further requirements to be able to use this module,
    see :ref:`Requirements <ansible_collections.zscaler.ziacloud.zia_sub_cloud_module_requirements>` for details.

    To use it in a playbook, specify: :code:`zscaler.ziacloud.zia_sub_cloud`.

.. version_added

.. rst-class:: ansible-version-added

New in zscaler.ziacloud 1.0.0

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- Updates subcloud excluded data centers based on the specified cloud ID.
- Subclouds cannot be created or deleted via the API; only exclusions can be updated.
- Use the info resource :literal:`zia\_sub\_cloud\_info` to look up cloud IDs.

Requirements
------------
The below requirements are needed on the host that executes this module.

- Zscaler SDK Python can be obtained from PyPI \ `https://pypi.org/project/zscaler\-sdk\-python/ <https://pypi.org/project/zscaler-sdk-python/>`__

.. Notes

Notes
-----

.. note::
   - Check mode is supported.
   - :literal:`cloud\_id` is required to identify the subcloud.
   - :literal:`state=absent` is a no\-op; subclouds cannot be deleted via the API.

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    - name: Update subcloud exclusions
      zscaler.ziacloud.zia_sub_cloud:
        provider: '{{ provider }}'
        cloud_id: 31649
        exclusions:
          - datacenter:
              - id: 5313
                name: "DC Name"
            country: "United States"
            end_time: 1735689600

    - name: Update subcloud with end_time_utc
      zscaler.ziacloud.zia_sub_cloud:
        provider: '{{ provider }}'
        cloud_id: 31649
        exclusions:
          - datacenter:
              - id: 5313
            country: "United States"
            end_time_utc: "01/15/2025 12:00:00 pm"
